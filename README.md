⭐ Patrick AI WhatsApp Bot

Bot WhatsApp multi-fitur berbasis Node.js yang dilengkapi AI, downloader, tools, games, sticker, group management, dan berbagai fitur lainnya.

✨ Features

- AI Chat
- Downloader
- Sticker Maker
- Group Management
- Search Tools
- Games
- Media Converter
- Owner Commands
- Utilities
- Dan banyak fitur lainnya

📦 Installation

Clone repository:

git clone https://github.com/rexz12/patrick-ai-whatsapp-bot
cd patrick-ai-whatsapp-bot

Install dependencies:

npm install

Jalankan bot:

npm start

atau

node .

⚙️ Configuration

Edit file konfigurasi sesuai kebutuhan:

config.js

Sesuaikan:

- Nomor Owner
- Nama Bot
- API Key
- Dan konfigurasi lainnya

🖥️ Pterodactyl

Startup Command:

npm start

Disarankan menggunakan:

- Node.js 20+
- RAM 1GB+

📋 Requirements

- Node.js 18+
- WhatsApp Account
- Internet Connection
